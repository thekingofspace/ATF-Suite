# ATF Suite

Open source tools for making your own ATF game.

ATF Suite is a collection of self-contained modules you can drop into a Roblox
project and build on. Everything is plain Luau and replicated via Rojo, so you
own the code and can extend any part of it.

## API

### PropPlacer

The prop / room toolkit — props, the objects that spawn them, the rooms (zones)
that hold them, and an in-game editor for transforming them.

- [Overview &amp; Usage](/api/propplacer.md)
- [Prop](/api/prop.md) — a single placed prop
- [PropObject](/api/propobject.md) — a reusable template that spawns props
- [Zone](/api/zone.md) — a room that groups and replicates props
- [ScaleManager](/api/scalemanager.md) — an in-game move / rotate / resize editor
